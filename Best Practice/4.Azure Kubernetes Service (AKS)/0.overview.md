# Core definition

- When you install the NGINX Ingress Controller, AKS creates ONE public IP. This is the entry point for your whole cluster.

- The way access to the Pod from the Internet

``` sh
User Browser
    ↓
DNS Provider (api.myapp.com → 20.205.33.10)
    ↓
Public IP (Ingress Controller)
    ↓
Ingress Rules (domain/path matching)
    ↓
Service (stable internal endpoint)
    ↓
Pod(s) (your application)
```
